# Copilot Spaces Quick Start for Team Superhelt

A 5-minute guide to start using Copilot Spaces with the [navikt/historisk-superhelt](https://github.com/navikt/historisk-superhelt) repository.

## Setup (5 minutes)

### 1. Create Your Space

1. Go to [github.com/copilot/spaces](https://github.com/copilot/spaces)
2. Click **Create Space**
3. Name: `Team Superhelt Knowledge Hub`
4. Click **Save**

### 2. Add the Repository

1. Click **Add sources**
2. Type: `navikt/historisk-superhelt`
3. Select the repository
4. Click **Add**

### 3. Add Instructions

Click **Instructions** and paste:

```markdown
## Team Superhelt Context
- Repository: navikt/historisk-superhelt
- Purpose: Team onboarding and collaboration
- When answering, reference specific files and explain reasoning
```

Click **Save**.

## Your First Questions

Try these prompts to get started:

### Understand the Project
```
What does the historisk-superhelt project do? 
Provide an overview of its purpose and main features.
```

### Explore the Codebase
```
What is the structure of this repository? 
What are the main folders and what do they contain?
```

### Setup Your Environment
```
How do I set up my local development environment? 
What dependencies do I need to install?
```

### Learn Team Practices
```
What is the branching strategy and pull request process 
used in this repository?
```

## Collaboration Tips

### Ask Better Questions
- ✅ **Specific**: "How does authentication work in src/auth/login.ts?"
- ❌ **Vague**: "How does authentication work?"

### Create Issues
```
Attach issue: @navikt/historisk-superhelt/issues/5
Help me add context to this issue based on the codebase.
```

### Share Knowledge
- Invite team members to your space
- Create documentation based on conversations
- Use spaces for code reviews and discussions
